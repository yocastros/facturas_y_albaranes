Información OCR
¿Cómo funciona?
1. Sube la factura o albarán
2. El OCR extrae los datos automáticamente
3. El sistema busca albaranes coincidentes
4. Se realiza el neteo automático si hay coincidencia
Criterios de neteo:
✓ Mismo proveedor (CIF)
✓ Fecha dentro de ±30 días
✓ Nº albarán mencionado en factura (prioritario)
Formatos soportados:
PDF · PNG · JPG · TIFF · BMP
